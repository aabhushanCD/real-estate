# Real Estate Dashboard App

A clean and minimal real estate application where users can sign up, log in, view properties, and manage favorites.

---

## Tech Stack

- **Frontend:** React, Tailwind CSS, React Router, TanStack Query  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB (Mongoose)  
- **Authentication:** JWT with HttpOnly cookies  
- **Notifications:** Sonner (Toast)

---

## Flow

dashboard -> signup -> login -> property add / remove / view
